# 🌿 Karunada-Vanya

### Wildlife Awareness & Forest Conservation Android Application

Karunada-Vanya is a modern Android application developed as part of the VTU Internship 2026 – Android App Development using Generative AI.

The application promotes wildlife awareness, forest conservation, environmental education, and human–wildlife coexistence across Karnataka through an interactive Android platform powered by Firebase and WebView integration.

The project combines Android development using Kotlin with modern web technologies to create a responsive wildlife awareness platform accessible directly on Android devices.

---

# 📖 Project Overview

Karunada-Vanya is designed to help users:

* Report wildlife sightings
* Learn about Karnataka wildlife
* Explore forest conservation content
* Receive wildlife alerts and updates
* Promote environmental awareness

The application integrates a web-based wildlife platform inside a native Android application using WebView technology.
